## 함수 선언식  
~~~ javascript
function a(){
	// ...
}

a();
~~~

## 함수 표현식
~~~ javascript
var a = function aa(){
	// named function expression
	...
}

var b = function() {
	// annonymous function expression
	...
}
~~~


아래와 같이 표현하는 것을 권장한다.  

함수 선언식은 호이스팅에 영향을 받는다.  
협업 도중 같은 이름의 function이 함수 선언식으로 표현할 경우 마지막에 끌어올려진 것으로 지정이 된다.
